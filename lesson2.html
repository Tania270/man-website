Урок 2: Основи CSS

CSS — це мова стилів, яка дозволяє змінювати вигляд HTML-елементів.

Що можна робити з CSS:

  • Змінювати кольори
  • Додавати відступи
  • Змінювати шрифти
  • Робити сайти красивими й зручними

Приклад CSS-коду:

body {
  background-color: #f0f8ff;
  font-family: sans-serif;
}
h1 {
  color: darkblue;
}
